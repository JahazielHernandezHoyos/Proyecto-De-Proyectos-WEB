Controlar Remotamente

Ahora para manejar la raspberry remotamente lo podremos hacer con una aplicación llamada vnc para saber la ip lo podremos hacer en la raspberry abriendo una terminal y colocando en el mismo “ifconfig” sin comillas

la ip aparece en la segunda línea generalmente y empieza con 192.168 como se muestra en la imagen también se puede hacer en windows si no se tiene accesibilidad a la misma mediante el comando “ping raspberrypi.local” sin comillas en el símbolo de sistema de windows que se puede abrir dándole al buscador y escribir cmd o simbolo de sistema la tercera forma es entrar al router y verificar la ip que se podrá ver en próximos post una vez obtenida la ip del raspberry vamos a descargar un programa llamado VNC que también se puede Descargar

VNC

https://www.realvnc.com/es/connect/download/viewer/ una vez descargado lo instalan de manera similar al programa raspbian y una vez esté instalado van al buscador de windows buscan el VNC y lo abren, les aparecera una ventana desmarcar la casilla y continúan una vez dentro del programa irán a la siguiente ubicación Archivo>Nueva conexión una vez estando ahi colocas la ip de la raspberry donde dice VNC server en mi caso la ip es 192.168.0.33 si siguieron los pasos deberian tener algo asi al final:
